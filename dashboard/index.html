Mathematical Proof & Transformation Method

Direct optimization of the curve is difficult because we have data coordinates $(x_i, y_i)$ but no matching parameter value $t_i$. We decouple the equations and solve for $t_i$ analytically using a 2D coordinate rotation transformation.

1. Coordinate Transformation

We shift the coordinates to eliminate $X$ and the constant offset $42$:

x̃ = x - X
ỹ = y - 42

This leaves us with the simplified system:

x̃ = t·cos(θ) - e^(M·t)·sin(0.3t)·sin(θ)
ỹ = t·sin(θ) + e^(M·t)·sin(0.3t)·cos(θ)

2. Isolating Parameter t

By rotating coordinates through $\theta$, we project the position vector onto the longitudinal axis, cancelling out the transcendental term:

t = x̃·cos(θ) + ỹ·sin(θ)

Thus, for any candidate values of $\theta$ and $X$, we calculate the exact $t_i$ for each dataset point directly:

t_i = (x_i - X)·cos(θ) + (y_i - 42)·sin(θ)

3. Transverse Fitting

Similarly, projecting onto the perpendicular transverse axis yields:

-x̃·sin(θ) + ỹ·cos(θ) = e^(M·t)·sin(0.3t)

This gives our objective function to minimize over $\theta$, $M$, and $X$:

Loss = Mean(|Actual_z - Expected_z|)
